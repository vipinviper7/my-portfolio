# vipinfolio.in

Personal portfolio website for Vipin — Visual Designer at Zoho.

## Live

[vipinfolio.in](https://vipinfolio.in)

## Pages

- **Home** — Hero, tabbed works section (Case Studies, Visual Design, Vibe Coded), contact footer
- **Experience** — Interactive scroll timeline with accordion dropdowns, education section
- **Case Study 1** — Recent Updates feature for Zoho Sprints
- **Case Study 2** — Settings screen redesign for Zoho Sprints

## Tech Stack

- HTML, CSS, vanilla JS — no frameworks
- Google Fonts (Space Grotesk, Syne)
- Hosted on GitHub Pages with custom domain
